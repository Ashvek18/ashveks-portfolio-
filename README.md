## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Ashvek18/ashvek-padwal-portfolio.git
```

Navigate to project directory:

```bash
cd ashvek-padwal-portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
